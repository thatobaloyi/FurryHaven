Getting Started +

Welcome to the Admin Dashboard! Here’s what you need to know to navigate efficiently:

  • Use the sidebar to access modules like Users, Reports, Analytics, Notifications, and Settings.
  • Hover over items in the sidebar to see descriptions of each module.
  • Always log out when done to secure your account.

User Management +

  • Add Users: Fill in all required fields and assign appropriate roles.
  • Edit Users: Update user info, roles, or permissions.
  • Remove Users: Soft delete users when necessary. Deleted users can be restored if needed.
  • Reset Passwords: Use this to help users regain access to their accounts.

Data Management +

  • View, edit, or delete records such as boarding animals or reports.
  • Export data for reporting or backups.
  • Archived or deleted records can be restored from the “Deleted Records” module.

Analytics & Reporting +

  • Monitor charts and statistics to track trends in boarding, health, or monthly activities.
  • Export reports for internal use or presentations.
  • Ensure data accuracy before sharing reports externally.

Troubleshooting +

  • Dashboard not loading: Refresh the page or clear browser cache.
  • Missing modules: Check your admin role permissions.
  • Error when approving/deleting records: Ensure the database connection is active.
